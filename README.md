#  Steel Industry Energy Consumption Analysis & Forecasting

## Overview
Applied Business Intelligence and Predictive Modeling to analyze 
electricity consumption patterns in the steel manufacturing industry 
and forecast future energy usage to support data-driven decisions.

## Key Results
- Built an interactive Power BI dashboard to monitor energy KPIs 
  and identify operational inefficiencies
- Revealed 99% correlation between CO₂ emissions and energy consumption
- Cleaned dataset from 28,032 to 20,384 rows using IQR outlier removal
- Random Forest model achieved 91.8% classification accuracy across load types
- Multiple Linear Regression forecasting model reached 97.49% accuracy (R² = 0.9746)

## Tools & Technologies
- Python (Pandas, NumPy) — data preprocessing & ETL
- Power BI — dashboard & KPI reporting
- Orange Data Mining — machine learning classification
- R Programming — statistical forecasting

## Methodology
1. **Data Cleaning** — IQR outlier removal, one-hot encoding, Z-score standardization
2. **Classification** — Decision Tree, SVM, Logistic Regression, Random Forest
3. **Forecasting** — Multiple Linear Regression on CO₂, reactive power, NSM
4. **Dashboard** — Interactive Power BI report for management reporting
